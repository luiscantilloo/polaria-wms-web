# Módulo: warehouses

Configuración y gestión de almacenes.

## Responsabilidades

- CRUD de almacenes (bodegas, centros de distribución)
- Definición de zonas, pasillos, estantes y ubicaciones (bins)
- Capacidad y restricciones por ubicación
- Asignación de almacenes a empresas

## Funcionalidades previstas

- Mapa o vista jerárquica del almacén (zona → pasillo → ubicación)
- Crear y editar estructura de almacenamiento
- Definir tipos de ubicación (picking, reserva, recepción, despacho)
- Consultar ocupación por ubicación

## Entidades relacionadas

- Almacén, Zona, Pasillo, Ubicación, TipoUbicación
