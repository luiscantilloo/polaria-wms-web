# Módulo: transport

Logística, envíos y transporte.

## Responsabilidades

- Gestión de transportistas y vehículos
- Creación y seguimiento de guías de envío
- Planificación de rutas de entrega
- Confirmación de entregas

## Funcionalidades previstas

- Asignar paquetes a transportista o ruta
- Generar guía de remisión / etiqueta de envío
- Tracking de estado de envío (en tránsito, entregado, devuelto)
- Registro de entrega con firma o foto
- Costos de flete por envío

## Entidades relacionadas

- Transportista, Vehículo, GuíaEnvío, Ruta, Entrega
