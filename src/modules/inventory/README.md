# Módulo: inventory

Gestión de inventario y stock.

## Responsabilidades

- Catálogo de productos (SKU, descripción, unidades)
- Niveles de stock por almacén y ubicación
- Movimientos de inventario (entradas, salidas, transferencias, ajustes)
- Conteos cíclicos y conciliación de inventario
- Trazabilidad por lote y fecha de vencimiento

## Funcionalidades previstas

- Consulta de stock en tiempo real
- Ajustes manuales con motivo y auditoría
- Transferencias entre almacenes o ubicaciones
- Alertas de stock mínimo/máximo
- Historial de movimientos por producto

## Entidades relacionadas

- Producto, SKU, Stock, MovimientoInventario, Lote, Conteo
