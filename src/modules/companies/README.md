# Módulo: companies

Gestión de empresas y organizaciones.

## Responsabilidades

- CRUD de empresas (clientes, proveedores o la propia organización)
- Datos fiscales y de contacto
- Configuración por empresa (moneda, zona horaria, impuestos)
- Relación empresa ↔ almacenes ↔ usuarios

## Funcionalidades previstas

- Registro y edición de empresas
- Asociar usuarios a una o varias empresas
- Configuración de parámetros operativos por empresa
- Multi-tenant: aislamiento de datos por empresa

## Entidades relacionadas

- Empresa, Sucursal, ConfiguraciónEmpresa
