# Módulo: purchases

Órdenes de compra y recepción de mercancía.

## Responsabilidades

- Creación y seguimiento de órdenes de compra (OC)
- Recepción parcial o total de mercancía
- Vinculación con proveedores
- Generación de movimientos de inventario al recibir

## Funcionalidades previstas

- Crear OC con líneas de producto, cantidades y precios
- Flujo de aprobación de compras
- Registrar recepción contra OC (match cantidad/precio)
- Consultar OC pendientes de recepción
- Integración con módulo de inventario al confirmar recepción

## Entidades relacionadas

- OrdenCompra, LíneaCompra, Recepción, Proveedor
