# src/assets/icons/

Iconos de la interfaz de usuario.

## Propósito

Almacena iconos SVG o PNG que se usan en botones, navegación, estados y acciones del WMS.

## Convención

- Prefiere SVG para iconos escalables.
- Nombra los archivos en kebab-case: `warehouse.svg`, `arrow-left.svg`.
- Si se adopta una librería de iconos (Lucide, Heroicons), esta carpeta puede quedar para iconos personalizados de Polaria.
