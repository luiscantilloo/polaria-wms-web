# src/assets/images/

Imágenes de contenido de la aplicación.

## Propósito

Banners, ilustraciones, fotos de productos y cualquier imagen que se importe en componentes React.

## Convención

- Usa formatos WebP o PNG según necesidad de transparencia.
- Nombra en kebab-case: `empty-state-inventory.png`.
- Para imágenes de productos dinámicas (desde base de datos), usa URLs de Supabase Storage en lugar de esta carpeta.
