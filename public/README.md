# public/

Archivos estáticos servidos directamente por Next.js en la raíz del sitio.

## Propósito

Todo lo que coloques aquí es accesible públicamente sin pasar por el bundler. Por ejemplo, `/next.svg` se sirve desde `public/next.svg`.

## Contenido típico

- Favicon e iconos de la aplicación
- Imágenes de marca (logos, ilustraciones)
- Archivos `robots.txt`, `sitemap.xml`
- Fuentes locales (si no se usan Google Fonts)

## Convención

Usa esta carpeta solo para recursos que deban tener una URL fija y pública. Para assets importados en componentes, prefiere `src/assets/`.
